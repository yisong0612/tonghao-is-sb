Project Overview
This project is a Node.js + Express + MySQL based campus secondhand trading website, designed specifically for campus scenarios. It enables students to post, browse, and manage secondhand items, facilitating efficient circulation of idle goods within the campus.
Core Features
User System: User registration/login, personal information management
Item Management: Post, edit, remove, and filter secondhand items by category
Item Browsing: Item list display, detail view, and search functionality
Transaction Communication: Item comments/direct messages (adjust based on actual features)
Frontend Pages: Responsive design, optimized for desktop campus use

Technology/Tool	    Purpose
Node.js	            Backend runtime environment
Express.js	        Backend web framework for handling API requests
MySQL	            Database for storing user, item, and transaction data

 Local Setup & Run
1. Prerequisites
Install MySQL (v5.7+ / v8.0+ supported) and start the MySQL service

2. Clone the Repository
git clone https://github.com/yisong0612/cp3407
cd cp3407

3. Install Dependencies
npm install

4. Database Configuration
   1.Open MySQL and create a new database (e.g., campus_secondhand)
    CREATE DATABASE campus_secondhand CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   2.Execute the CP3407_database.sql script to initialize tables and data
    # Run via command line (replace with your MySQL username and database name)
    mysql -u your-mysql-username -p campus_secondhand < CP3407_database.sql
   3.Update the database connection settings in db.js (username, password, database name, host)
    // Example configuration in db.js
    const mysql = require('mysql');
    const connection = mysql.createConnection({
      host: 'localhost',
      user: 'root', // Your MySQL username
      password: 'your-mysql-password',
      database: 'campus_secondhand' // Your created database name
    });

5. Start the Project
    node server.js
    # Or if nodemon is configured: npm start

6. Access the Application
    Open your browser and navigate to http://localhost:3000 (port number depends on your server.js configuration).


Project Notes
This project is designed for campus secondhand trading scenarios. You can extend features as needed (e.g., order system, payment integration, admin backend).
The database schema is defined in CP3407_database.sql, including core tables like users and items.
Frontend pages are located in the public folder, which can be modified directly for style and interaction updates.
